▴ Qualcomm DIAG over USB

The Qualcomm DIAG interface exposes /dev/diag as a USB serial port to the host. When active, tools such as QCSuper, QXDM, and QPST can connect via USB cable. This toggle adds or removes diag from the live USB gadget functions without changing the composition mode. Effective immediately — USB briefly reconnects.

DIAG: loading…
Expose DIAG port to USB host
⚠ Applying this toggle briefly disconnects USB (RNDIS + ADB). The browser tab may hang during the drop — this is normal. Refresh after ~10 seconds if the page stops responding. Only applies to the live state; the saved composition mode is unchanged.

USB Composition Mode

Selects the USB gadget personality from /data/usb/boot_hsusb_composition. The saved mode persists across reboots via /usrdata/mode.cfg. "Next Boot Only" writes /usrdata/mode_tmp.cfg (one-shot; consumed at boot). "Apply Now" writes sysfs directly — USB drops for ~2 seconds.
